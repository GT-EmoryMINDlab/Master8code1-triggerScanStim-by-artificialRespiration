Triggering for rodent fMRI - Ventilation - Scan - Stimulation

(Version : w202102)

  1. Set animal artificial ventilation 1.6Hz and trigger out to Master-8 EXT1
  2. ALL-1-ENTER (call program #1)
  3. Set ch1 as TRIG mode: (triggered by EXT1, switch bar to control input, ‘off’ for resting state scan)
  1. TRIG-1-ENTER
  2. INTER-1-100-ENTER-3-ENTER (100ms interval, i.e. 10Hz)
  3. DURA-1-10-ENTER-3-ENTER (10ms pulse, single pulse @ TRIG mode)
  1. CONCT-1-2-ENTER (ch1 connects ch2 to trigger ch2)
  2.  TRAIN-2-ENTER (set ch2 as TRAIN mode for TR = 1.25sec EPI scans: 10min+12.5sec dummy scan; ventilation rate set on 1.6Hz, i.e. phase-locked to every other breath)
  1. INTER-2-1250-ENTER-3-ENTER (1.25sec cycle)
  2. DURA-2-10-ENTER-3-ENTER (10ms pulse)
  3. M-2-510-ENTER-0-ENTER ( 500 scans + 10 dummy scans - need setting up in scan, ALSO switch ch2 to “-” for negative pulse to trigger Bruker scanner)
  1. CONCT-1-8-ENTER ( ch1 connects ch8 to trigger ch8,)
  2. TRIG-8-ENTER ( ch8 as TRIG mode to set delay, no delay in TRAIN mode)
  1. DELAY-8-12.5-ENTER-0-ENTER (delay 12.5s during 10 dummy scans)
  2. INTER-1-100-ENTER-3-ENTER (100ms interval, i.e. 10Hz)
  3. DURA-1-10-ENTER-3-ENTER (10ms pulse, single pulse @ TRIG mode)
  1. CONCT-8-5-ENTER ( ch8 connects ch5 to trigger ch5,)
  2. TRAIN-5-ENTER (Ch5 for stim. block design,10s-on 20s-off for 10min scans)
  1. INTER-5-30-ENTER-0-ENTER (30s per cycle)
  2. DURA-5-10-ENTER-0-ENTER (10s stim. per cycle)
  3. M-5-20-ENTER-0-ENTER (20 cycles)
  1. CT-5-3-ENTER (ch5 connects ch3 to trigger ch3)
  2. TRAIN-3-ENTER (ch3 for stimulation)
  1. M-3-40-ENTER-0-ENTER (4Hz lasting 10sec, i.e.10*4=40 pulses)
  2. INTER-3-250-ENTER-3-ENTER (1/4=250ms per cycle)
  3. DURA-3-10-ENTER-3-ENTER (10ms pulses, setting ISO-flex 1.5mA for forepaw)
